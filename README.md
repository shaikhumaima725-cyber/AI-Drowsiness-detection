AI Drowsiness Detection is a real-time driver-fatigue monitoring system that uses only a standard webcam — no wearables, no specialized hardware. It tracks three key fatigue signs: closed eyes (via Eye Aspect Ratio), yawning (via Mouth Aspect Ratio), and head nodding (via head-pose estimation), using MediaPipe Face Mesh and OpenCV. A lightweight state machine combines these signals with consecutive-frame and rolling-window checks, so a single blink never false-alarms, while genuine drowsiness triggers an instant visual and audible alert within about a second.
The repo includes the full Python webcam app, a no-install browser demo with a live siren alert, and an offline simulation script for testing without a camera. Every threshold is a clearly named, tunable variable — fully explainable, no black-box model. Runs in real time on CPU alone, no GPU or cloud required.
